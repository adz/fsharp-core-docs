NativePtr Module

Namespace: FSharp.NativeInterop

Assembly: FSharp.Core.dll

Contains operations on native pointers. Use of these operators may result in the generation of unverifiable code. Library functionality for native interopability. See also F# External Functions in the F# Language Guide.
